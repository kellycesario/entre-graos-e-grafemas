# Sobre o Projeto

Este é um projeto desenvolvido com **Next.js 13** utilizando o **App Router**, integrado com o **Contentful** como CMS e hospedado na **Vercel**. A estrutura é projetada para fornecer desempenho e escalabilidade, aproveitando as melhores práticas de desenvolvimento frontend moderno.
